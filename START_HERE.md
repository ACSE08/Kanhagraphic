# 🐛 **QUICK TROUBLESHOOTING**

### **"Can't connect from phone"**
- Make sure both on same WiFi
- Use full URL: `http://192.168.1.2:3000` (include :3000)
- Try hard refresh on phone

### **"Server won't start"**
- Run: `taskkill /IM node.exe /F`
- Then: `npm run dev`

### **"Port already in use"**
- Try: `npm run dev -- --port 3001`

### **"Styles look wrong on phone"**
- Refresh: `Ctrl+Shift+R` on desktop, or swipe down on phone
- Clear cache in Settings
