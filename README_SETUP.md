## 🆘 **QUICK TROUBLESHOOTING**

| Problem | Solution |
|---------|----------|
| Port 3000 in use | `taskkill /IM node.exe /F` |
| Can't connect from phone | ✅ Same WiFi? ✅ Right IP? ✅ Include :3000? |
| Server won't start | Run: `npm install` → `npm run dev` |
| Styles not loading | Hard refresh: `Ctrl+Shift+R` |
| Server stopped | Just start again with `npm run dev` |
